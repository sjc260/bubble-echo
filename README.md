# Bubble Echo

> *Custodisci i tuoi momenti più belli con la voce.*

**Allegato B** — Prototipo visivo UX dell'app iOS *Bubble Echo*.

Bubble Echo è un'app di **voice journaling emotivo** che custodisce i momenti positivi quotidiani sotto forma di bolle colorate. Ogni bolla raccoglie una registrazione vocale, un'emozione e i suoi metadati: si può riascoltare, condividere e ritrovare nel proprio universo personale.

Il prototipo poggia su tre costrutti psicologici:

- **Savouring** — assaporare il momento positivo nel mentre lo si vive
- **Positive Mental Time Travel (PMTT)** — rivisitare emozioni passate per nutrire il presente
- **Preservazione condivisa della memoria emotiva** — donare una bolla a chi ci è caro
